# IMAGENES REQUERIDAS - ATILAX Website

Este documento lista todas las imagenes necesarias para el sitio web de ATILAX, organizadas por seccion con descripciones detalladas para facilitar su creacion con herramientas de IA como Nano Banana, Midjourney, DALL-E o similares.

---

## INSTRUCCIONES GENERALES PARA GENERACION CON IA

### Estilo Visual
- **Estilo**: Fotografico profesional, industrial, corporativo
- **Colores predominantes**: Azul corporativo (#1E3A5F), naranja/ambar (#F59E0B), tonos neutros
- **Iluminacion**: Natural o industrial, bien iluminado
- **Resolucion sugerida**: 1920x1440px (aspect ratio 4:3) o 1920x1080px (16:9)
- **Formato**: JPG para fotos, PNG para graficos con transparencia

### Contexto Venezuela
- Paisajes y ambientes tipicos de Venezuela
- Infraestructura reconocible del pais
- Personas con apariencia latinoamericana cuando aplique
- Vegetacion tropical en exteriores
